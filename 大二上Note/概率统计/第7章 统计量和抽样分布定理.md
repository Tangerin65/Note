# 统计量
统计量，就是从样本数据中计算出来的一个“数字总结”，本质作用是以小见大
样本->样本函数=统计量->观测值
统计量**不含任何未知参数**
统计量是随机变量，其观测值是实数

# 抽样分布定理
**回答”当我从总体中反复抽取样本并计算一个统计量时，这些统计量本身会形成什么样的分布“这个问题**
#### 1.正态分布下总体的样本均值分布
![[Chapter7剩余部分教材.pdf#page=3&rect=21,480,338,572|Chapter7剩余部分教材, p.3]]
![[Chapter7剩余部分教材.pdf#page=3&rect=49,273,360,325|Chapter7剩余部分教材, p.3]]
 - 样本均值服从(非标准)正态分布
 - 样本均值/方差齐与总体均值/方差有关
#### 2.正态分布总体下的样本方差分布(卡方分布)
关注标准化后的样本方差。
![[Chapter7剩余部分教材.pdf#page=3&rect=49,182,355,267|Chapter7剩余部分教材, p.3]]

![[Chapter7剩余部分教材.pdf#page=3&rect=51,105,357,171|Chapter7剩余部分教材, p.3]] 
 - $\overline{X}$和$S^2$独立，但$\overline{X}$和$B^2$ 不独立，这是统计量独立**唯一的突破口**
 - 样本方差乘以某个常数后服从卡方分布
